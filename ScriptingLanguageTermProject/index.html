About the Project

This website is designed as an online recipe book system. It allows users to browse various recipes, submit their own, explore categorized recipe data and let them have fun. The project includes five pages: a homepage, a recipe list, a submission form, a categorized recipe table, and a surprise page.

The system is designed for cooking lovers and aims to be dynamic, interactive, and user-friendly using JavaScript.
